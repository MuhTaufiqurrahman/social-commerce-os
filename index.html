📚 Cara Menggunakan Multi-Device Sync

✨ Multi-Device Sync

Fitur Baru! Sekarang Anda bisa sync data antar semua device:

  • 📱 Desktop → Kerja di kantor
  • 📱 iPad → Meeting dengan client
  • 📱 Mobile → Cek data di mana saja

Semua device sync otomatis setiap 30 detik! Data selalu up-to-date di semua device.

Quick Sync (Tanpa Setup)

  1. Pilih database → Klik Export
  2. Data ter-copy, Google Sheets terbuka
  3. Paste di Google Sheets (Ctrl+V)
  4. Selesai!

Multi-Device Auto Sync (Setup Sekali)

  1. Buat Google Sheets baru
  2. Extensions → Apps Script
  3. Copy script dari google-apps-script.js
  4. Deploy → Web app → Copy URL
  5. Paste URL di tab "Auto Sync"
  6. Enable Auto-Sync & Save
  7. Buka di device lain → Paste URL yang sama
  8. ✅ Semua device sync otomatis!

Tips Multi-Device

  • Setup sekali di 1 device, lalu paste URL yang sama di device lain
  • Sync interval 30 detik = real-time sync antar device
  • Offline mode: Tetap bisa kerja offline, sync saat online
  • Conflict resolution: Data terbaru selalu menang
  • Device tracking: Lihat device mana yang terakhir sync
  • Data aman: Tersimpan di Google Sheets (cloud backup)

Keamanan Data

  • Data tersimpan di Google Sheets Anda sendiri
  • Tidak ada server pihak ketiga
  • Anda kontrol penuh akses data
  • Backup otomatis di Google Drive
